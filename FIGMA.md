## Deskripsi Proyek
PromptKU adalah platform berbasis web yang dirancang untuk membantu kreator menemukan dan membagikan prompt Generative AI (seperti Midjourney, Stable Diffusion, dll). Website ini dikembangkan dengan pendekatan High-Fidelity dari desain Figma ke kode.

##  Fitur Utama
1.  Responsive Layout: Tampilan menyesuaikan Desktop dan Mobile (HP).
2.  Google Translate Widget: Fitur terjemahan (Inggris & Mandarin) yang terintegrasi rapi di header.
3.  Dark Mode / Light Mode: Mendukung pergantian tema visual.
4.  Galeri Interaktif: Filter kategori (Cloud/Local), Modal Popup detail, dan tombol Favorit.
5.  Formulir: Halaman Contact dan Kontribusi (Kritik & Saran) yang interaktif.

## Teknologi
- HTML5 (Semantik)
- CSS3 (Flexbox, Grid, Variables, Custom Google Translate Style)
- JavaScript (DOM Manipulation, Event Handling, LocalStorage)
